void NetProcGame.modes.Trough.launch_balls ( int  num,
Delegate  callback = null,
bool  stealth = false 
)

Either initiate a new launch or add another ball to the count of balls being launched. Make sure to keep a separate count for stealth launches that should not increase num_balls_in_play.

Parameters:
numNumber of balls to launch
callbackIf specified, the callback will be called once all balls are launched
stealthSet to true if the balls being launched should NOT be added to the number of balls in play. For instance, if a ball is being locked on the playfield, and a new ball is being used to keep 1 active ball in play, use stealth.
